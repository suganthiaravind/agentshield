# AgentShield — Quick Start
Follow this file top to bottom. The only thing you set is your repo path.
## 0 — Set your target
export REPO=/absolute/path/to/your-agent-repo
SET FIRST
## 1 — Install AgentShield
git clone https://github.com/your-org/agentshield.git pip install -e ".[semgrep,dev]" agentshield --version
INSTALL
## 2 — Run the Tier 1 scan
agentshield scan $REPO --scan-all-files
SCAN
## 3 — Tier 2 + Emulator (VS Code + Copilot)
Open $REPO in VS Code → paste SKILL.md → paste AGENT_EMULATOR_SKILL.md
## 4 — Generate report
agentshield merge $REPO --output-html report.html open report.html
MERGE
What each step does
Steps 0
Point at your repo
One variable. Every command below reads $REPO — set it once up front.
Step 1 — Install
AgentShield CLI installed
Clones the repo, creates a venv, bundles Semgrep. Takes ~60s once. Done when you see agentshield 4.x.x
Step 2 — Scan
Tier 1 static scan runs
70+ Semgrep rules fire against your code. Skill files for Copilot are written to .agentshield/
Step 3 — Copilot
Tier 2 + Emulator via VS Code
Paste two skill files into Copilot Chat. Copilot validates findings and simulates 14 adversary attack classes.
Step 4 — Report
Unified report generated
Merges all results into one HTML report — D/D/R findings, fix guidance, OWASP LLM, MITRE ATLAS, CWE.
